Бот резервного копирования для Telegram
Этот бот для Telegram позволяет создавать резервные копии файлов, отправленных ему. Бот может сохранять файлы локально и загружать их на Яндекс.Диск и Google Drive.

Установка
Установите необходимые пакеты, выполнив следующую команду:
pip install -r requirements.txt
Получите необходимые API-токены и учетные данные:

Токен Telegram-бота:

Создайте нового бота в Telegram, следуя инструкциям на официальной странице Telegram Bot API.
Получите API-токен для вашего бота.
Сохраните API-токен в файле с именем TGtoken.txt. Каждый бот должен иметь свой собственный уникальный токен.
Токен Яндекс.Диска:

Перейдите на Яндекс.Консоль для разработчиков и авторизуйтесь.
Создайте новое приложение и получите токен доступа к API Яндекс.Диска.
Сохраните токен в файле YDtoken.txt.
Учетные данные Google Drive:

Перейдите на Консоль разработчика Google и авторизуйтесь.
Создайте новый проект или выберите существующий.
Включите API Google Drive для вашего проекта.
Создайте учетные данные OAuth 2.0 и загрузите файл JSON с клиентскими секретами.
Сохраните файл JSON с клиентскими секретами с именем client_secrets.json.
Создайте папку с именем backup в том же каталоге, что и скрипт. В эту папку будут сохраняться резервные копии файлов.

Использование
Запустите бота, выполнев скрипт

Отправьте команду /start боту в Telegram, чтобы начать процесс резервного копирования.

По запросу, отправьте файлы, которые вы хотите сохранить в резервную копию. Вы можете отправить несколько файлов одновременно. Обязательно отправляйте файлы, даже если они являются фотографиями или другими мультимедийными объектами. При отправке с телефона следует использовать функцию прикрепления файлов.

После отправки всех файлов вам будет предложено загрузить их на Яндекс.Диск. Ответьте "Yes" или "No".

Затем вам будет предложено загрузить файлы на Google Drive. Ответьте "Yes" или "No".

После загрузки файлов на Яндекс.Диск и/или Google Drive, вам будет предложено продолжить процесс резервного копирования или вернуться в меню.
